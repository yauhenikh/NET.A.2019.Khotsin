Mini-project "File Cabinet"

Step 1

Work with records, use memory, List

"create" command
```
> create
First name: ...
Last name: ...
Date of birth: ...

```
"list" command
```
> list
#1, John, Doe
#2, Stan, Smith
```
"stat" command
```
> stat
3 records.
```
